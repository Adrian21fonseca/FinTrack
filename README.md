# 🚀 FinTrack

**FinTrack** es una aplicación web de gestión de finanzas personales diseñada para ser **simple, intuitiva y accesible para cualquier usuario**.

El objetivo principal es ayudar a las personas a **entender y controlar su dinero sin complicaciones**, a través de una interfaz clara y funcionalidades prácticas.

---

## 🧠 Filosofía del proyecto

Muchas herramientas financieras son complejas o difíciles de usar.
FinTrack nace con una idea clara:

> **Las finanzas deberían ser fáciles de entender para todos.**

Este proyecto forma parte de un proceso de aprendizaje enfocado en:

* Aplicar conocimientos reales de desarrollo web
* Construir productos funcionales desde cero
* Priorizar la experiencia de usuario (UX)

---

## 🎯 Objetivos

* Crear una aplicación funcional de control de ingresos y gastos
* Desarrollar una arquitectura escalable desde el inicio
* Implementar una interfaz moderna y responsive
* Evolucionar el proyecto hacia un producto real

---

## ⚙️ Tecnologías utilizadas

* HTML5
* CSS3 (Flexbox + Grid)
* JavaScript (Vanilla)
* LocalStorage (persistencia de datos inicial)

---

## 🧩 Funcionalidades (MVP)

* Registro de ingresos y gastos
* Visualización del balance total
* Lista de movimientos
* Edición y eliminación de registros
* Categorías básicas
* Persistencia de datos en el navegador

---

## 🚧 Funcionalidades futuras

* Sistema de autenticación (registro e inicio de sesión)
* Filtros por fecha y categoría
* Gráficas y análisis financiero
* Cálculo de porcentajes (ahorro, gastos, distribución)
* Simuladores financieros (créditos, hipotecas)
* Modo oscuro
* Integración con backend (Node.js + base de datos)

## 🏗️ Estructura del proyecto

```bash
fintrack/
│

│

│   ├── styles.css        # Estilos globales y variables (:root)

│   ├── app.js            # Punto de entrada

│
├── /assets
│   ├── images/
│   └── icons/
│
└── /pages
    ├── dashboard.html

```
